# TOOLS.md

## Verfügbare Tools im Container

- `exec` – Shell-Kommandos ausführen (git, npm, etc.)
- `read` – Dateien lesen
- `write` – Dateien schreiben
- `edit` – Dateien bearbeiten
- `apply_patch` – Patches anwenden
- `message` – Telegram-Nachrichten

## genesisWorld MCP

Der genesisWorld-MCP-Server läuft als HTTP-Endpoint und stellt Tools zur Verfügung, um auf CAS genesisWorld zuzugreifen. Die genesisWorld-spezifischen MCP-Tools sind über das OpenClaw-Gateway direkt verfügbar.
